Sudoku Strategies

Basic Techniques

  • Single Candidate: When only one number can go in a cell
  • Single Position: When a number can only go in one position in a row, column, or box
  • Elimination: Remove possibilities based on existing numbers

Advanced Techniques

  • Naked Pairs: Two cells in a unit with the same two candidates
  • Hidden Pairs: Two candidates that appear only in two cells of a unit
  • X-Wing: A pattern that allows elimination of candidates
