✅ Core Text Functions (for URLs / Web Data)

LEFT

Purpose:Extracts a given number of characters from the left side of a string. Great for identifying protocols (http/https).
Return value:Text string
Syntax:LEFT(text, [num_chars])
Examples:=LEFT("https://domain.com", 8) // Returns "https://"

RIGHT

Purpose:Extracts a given number of characters from the right side of a string. Useful for finding file extensions (.html, .json).
Return value:Text string
Syntax:RIGHT(text, [num_chars])
Examples:=RIGHT("index.html", 5) // Returns ".html"

MID

Purpose:Extracts characters from the middle of a text string, given a starting position and length.
Return value:Text string
Syntax:MID(text, start_num, num_chars)
Examples:=MID("www.google.com", 5, 6) // Returns "google"

LEN

Purpose:Returns the number of characters in a text string. Useful for validating URL/Title lengths for SEO.
Return value:Numeric length
Syntax:LEN(text)
Examples:=LEN("https://toolshub365.com") // Returns 25

✅ Find & Extract (Important for URLs)

FIND

Purpose:Returns the starting position of one text string within another. Case-sensitive.
Return value:Numeric position
Syntax:FIND(find_text, within_text, [start_num])
Examples:=FIND("?", "page.php?id=5") // Returns 9

SEARCH

Purpose:Returns the starting position of a text string. Case-insensitive, supports wildcards (*, ?).
Return value:Numeric position
Syntax:SEARCH(find_text, within_text, [start_num])
Examples:=SEARCH("HTTP", "https://site.com") // Returns 1

TEXTBEFORE

Purpose:Returns text that occurs before a given character or string (delimiter). Ideal for stripping domains.
Return value:Text string
Syntax:TEXTBEFORE(text, delimiter, [instance_num])
Examples:=TEXTBEFORE("user@domain.com", "@") // "user"

TEXTAFTER

Purpose:Returns text that occurs after a given character or string. Great for getting URL slugs.
Return value:Text string
Syntax:TEXTAFTER(text, delimiter, [instance_num])
Examples:=TEXTAFTER("domain.com/page", "/") // "page"

TEXTSPLIT

Purpose:Splits text strings by using column and row delimiters. Replaces Text-to-Columns feature.
Return value:Spilled Array of Text
Syntax:TEXTSPLIT(text, col_delimiter, [row_delimiter])
Examples:=TEXTSPLIT("tag1,tag2,tag3", ",")

✅ Combine & Join Text

CONCATENATE

Purpose:Joins several text items into one text item (Legacy function).
Return value:Combined text string
Syntax:CONCATENATE(text1, [text2], ...)
Examples:=CONCATENATE("https://", A1, ".com")

CONCAT

Purpose:Combines the text from multiple ranges and/or strings (Modern version of CONCATENATE).
Return value:Combined text string
Syntax:CONCAT(text1, [text2], ...)
Examples:=CONCAT(A1:C1) // Joins cells A1, B1, C1

TEXTJOIN

Purpose:Combines text from multiple ranges/strings, separating them with a specified delimiter. Great for URL building.
Return value:Combined text string
Syntax:TEXTJOIN(delimiter, ignore_empty, text1, ...)
Examples:=TEXTJOIN("/", TRUE, "http:", "", "site.com", "api")

✅ Replace & Clean Web Data

SUBSTITUTE

Purpose:Replaces existing text with new text in a string. Excellent for fixing URL encoded characters (like %20).
Return value:Modified text string
Syntax:SUBSTITUTE(text, old_text, new_text, [instance_num])
Examples:=SUBSTITUTE("first%20name", "%20", " ")

REPLACE

Purpose:Replaces characters within text based on a specific starting point and length.
Return value:Modified text string
Syntax:REPLACE(old_text, start_num, num_chars, new_text)
Examples:=REPLACE("http://site", 1, 4, "https")

TRIM

Purpose:Removes all spaces from text except for single spaces between words. Essential for imported web data.
Return value:Cleaned text string
Syntax:TRIM(text)
Examples:=TRIM(" email@domain.com ")

CLEAN

Purpose:Removes all non-printable characters from text (line breaks, tabs often found in scraped data).
Return value:Cleaned text string
Syntax:CLEAN(text)
Examples:=CLEAN(A1) // Strips hidden web characters

✅ Case & Format

LOWER

Purpose:Converts text to lowercase. Crucial for normalizing email addresses and URLs.
Return value:Lowercase text
Syntax:LOWER(text)
Examples:=LOWER("USER@Domain.com") // user@domain.com

UPPER

Purpose:Converts text to uppercase. Used for formatting country codes, currency, etc.
Return value:Uppercase text
Syntax:UPPER(text)
Examples:=UPPER("usd") // USD

PROPER

Purpose:Capitalizes the first letter in each word of a text value. Good for name formatting.
Return value:Title case text
Syntax:PROPER(text)
Examples:=PROPER("john doe") // John Doe

EXACT

Purpose:Checks to see if two text values are exactly identical, including case (case-sensitive).
Return value:TRUE or FALSE
Syntax:EXACT(text1, text2)
Examples:=EXACT("TokenA", "tokena") // FALSE

✅ Conversion Functions

VALUE

Purpose:Converts a text string that represents a number to a number. Helpful for fixing CSV string numbers.
Return value:Numeric value
Syntax:VALUE(text)
Examples:=VALUE("1,000") // 1000

NUMBERVALUE

Purpose:Converts text to number in a locale-independent manner. Very useful for international web data.
Return value:Numeric value
Syntax:NUMBERVALUE(text, [decimal_separator], [group_separator])
Examples:=NUMBERVALUE("1.200,50", ",", ".") // 1200.5

TEXT

Purpose:Formats a number and converts it to text. Often used to prepare dates/numbers for API query strings.
Return value:Formatted text string
Syntax:TEXT(value, format_text)
Examples:=TEXT(TODAY(), "yyyy-mm-dd") // "2023-10-31"

✅ Encoding / Character Functions

CHAR

Purpose:Returns the character specified by the code number (ASCII). Useful for injecting line breaks or quotes.
Return value:Single Character
Syntax:CHAR(number)
Examples:=CHAR(34) // Returns double quote (")
=CHAR(10) // Returns line break

CODE

Purpose:Returns a numeric code for the first character in a text string. Helpful to identify hidden characters.
Return value:Integer (ASCII value)
Syntax:CODE(text)
Examples:=CODE("A") // Returns 65

UNICHAR

Purpose:Returns the Unicode character referenced by the given numeric value. Ideal for web symbols.
Return value:Unicode Character
Syntax:UNICHAR(number)
Examples:=UNICHAR(9745) // Returns ☑ (Checkbox)

UNICODE

Purpose:Returns the number (code point) corresponding to the first character of the text.
Return value:Integer
Syntax:UNICODE(text)
Examples:=UNICODE("☑") // Returns 9745

✅ Lookup for Web Data

XLOOKUP

Purpose:Modern lookup to match web IDs/URLs with data tables. Returns an exact match by default.
Return value:Matched Value / Array
Syntax:XLOOKUP(lookup, lookup_arr, return_arr)
Examples:=XLOOKUP(A2, WebData!A:A, WebData!B:B)

INDEX

Purpose:Retrieves a value from a grid of imported web data based on row/column numbers.
Return value:Cell Value / Reference
Syntax:INDEX(array, row_num, [column_num])
Examples:=INDEX(A1:C100, 5, 2)

MATCH

Purpose:Finds the position of an item (like a scraped tag) in a list. Often paired with INDEX.
Return value:Integer (Position)
Syntax:MATCH(lookup_value, lookup_array, [match_type])
Examples:=MATCH("keyword", A1:A100, 0)

FILTER

Purpose:Dynamically filters scraped datasets to show only rows matching specific criteria (e.g., Status = 200).
Return value:Spilled Array
Syntax:FILTER(array, include, [if_empty])
Examples:=FILTER(A2:C100, B2:B100="Live", "None")

✅ Logical / Error Handling (Web Data Cleaning)

IF

Purpose:Performs logic tests (e.g., checking if a URL starts with HTTPS).
Return value:Defined Value
Syntax:IF(logical_test, value_if_true, [value_if_false])
Examples:=IF(LEFT(A1,5)="https", "Secure", "Unsecure")

IFERROR

Purpose:Handles #VALUE! or #DIV/0! errors resulting from broken data imports or splits.
Return value:Value or Alternative
Syntax:IFERROR(value, value_if_error)
Examples:=IFERROR(VALUE(A1), 0)

IFNA

Purpose:Specifically handles missing lookup values (e.g., API ID not found).
Return value:Value or Alternative
Syntax:IFNA(value, value_if_na)
Examples:=IFNA(VLOOKUP(A1, B:C, 2, 0), "Not Found")

ISNUMBER

Purpose:Checks if parsed internet data evaluates as a number. Useful in FIND/SEARCH formulas.
Return value:TRUE or FALSE
Syntax:ISNUMBER(value)
Examples:=ISNUMBER(SEARCH("spam", A1))

ISTEXT

Purpose:Validates if extracted data is text (checking for formatting bugs during import).
Return value:TRUE or FALSE
Syntax:ISTEXT(value)
Examples:=ISTEXT(A1)

✅ Dynamic Array (Modern Excel Web Use)

SORT

Purpose:Automatically sorts lists of keywords or links imported from the web.
Return value:Spilled Sorted Array
Syntax:SORT(array, [sort_index], [sort_order])
Examples:=SORT(UNIQUE(A2:A100))

SORTBY

Purpose:Sorts a dataset by metrics (e.g., sorting URLs by their corresponding Pageviews).
Return value:Spilled Sorted Array
Syntax:SORTBY(array, by_array1, [sort_order1])
Examples:=SORTBY(A2:A10, B2:B10, -1)

UNIQUE

Purpose:Extracts deduplicated lists (like unique IPs or domains) from massive web dumps.
Return value:Spilled Unique Array
Syntax:UNIQUE(array, [by_col], [exactly_once])
Examples:=UNIQUE(A2:A5000)

SEQUENCE

Purpose:Generates numeric arrays. Excellent for building sequential API endpoints or pagination parameters.
Return value:Spilled Numeric Array
Syntax:SEQUENCE(rows, [columns], [start], [step])
Examples:="site.com/page=" & SEQUENCE(5)

✅ Workbook / Link Related

HYPERLINK

Purpose:Creates a clickable link to open a web page, file, or email address.
Return value:Clickable UI Link
Syntax:HYPERLINK(link_location, [friendly_name])
Examples:=HYPERLINK("https://toolshub365.com", "Visit Tool")

INDIRECT

Purpose:Constructs cell references from text strings. Can be used to dynamically map imported web tables.
Return value:Cell Value
Syntax:INDIRECT(ref_text, [a1])
Examples:=INDIRECT("A" & ROW())

🔥 Bonus Web / Internet Use Cases

Extract Domain Name (MID + FIND / TEXTBEFORE)

Purpose:Grabs the root domain from a full URL, removing protocols and paths.
Return value:Domain text (e.g. domain.com)
Syntax:TEXTBEFORE(TEXTAFTER(url, "://"), "/")
Examples:=TEXTBEFORE(TEXTAFTER("https://site.com/page", "://"), "/")

Extract URL Parameters (TEXTAFTER)

Purpose:Isolates UTM tags or query strings from a URL for tracking analysis.
Return value:Query string text
Syntax:TEXTAFTER(url, "?")
Examples:=TEXTAFTER("site.com?utm=123", "?") // "utm=123"

Clean Imported Data (TRIM + CLEAN)

Purpose:Strips unseen characters and excessive spaces from web-scraped data simultaneously.
Return value:Clean text
Syntax:TRIM(CLEAN(text))
Examples:=TRIM(CLEAN(A1))

Split CSV/Web Data (TEXTSPLIT)

Purpose:Separates comma or pipe delimited strings dumped from APIs into separate cells.
Return value:Array of cells
Syntax:TEXTSPLIT(text, delimiter)
Examples:=TEXTSPLIT(A1, ",") // Expands horizontally

Build Dynamic Links (HYPERLINK)

Purpose:Combines a base URL with cell variables (like search queries) to generate active links.
Return value:Clickable URL
Syntax:HYPERLINK("url_base" & cell, friendly_name)
Examples:=HYPERLINK("https://google.com/search?q=" & A1, "Search")
