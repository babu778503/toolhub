User Manual

Welcome to the KPI Calculator!

This tool is a self-contained HTML file. All data is saved directly in your browser's local storage, meaning it's private and works offline. No data is sent over the internet.

1. General Configuration

  1. Start/End Date: Set the period you want to track. The tool automatically defaults to the current month.
  2. Holidays: Click the üìÖ icon to open a calendar. Click on dates to mark them as holidays. They will not be counted as working days.
  3. Weekend Days: Check the boxes for days that should be considered weekends. These are also not counted as working days.

2. KPI & Target Configuration

  1. Add/Remove KPI: Use the + Add Another KPI button to add a new performance indicator. Use the √ó button on a KPI to remove it.
  2. Target Mode:
    • Monthly: Set a single, total target for the entire month.
    • Block Wise: This mode allows you to create custom sales blocks.
      • Click + Add Block to add another sales period.
      • The name of each block (e.g., "Block 1") is also editable. Just click on it and type a new name.
      • Set the End Date for each block (except the last one).
      • Set the Target % for each block (except the last one). The final block's percentage is calculated automatically to ensure the total is 100%.
      • Click the √ó on a block to remove it (you must have at least two).

3. Daily Actuals Tracking

In the main table, enter your achieved numbers for each KPI on the corresponding day. The dashboards and summaries will update instantly. When the tracker is Locked, you can only enter data for the current day. On mobile, only today's card is shown by default; use the "Show All Dates" button to see others.

4. Footer Actions

  1. Toggle Footer: Click the eye icon in the top-right of the header to hide or show the entire "Actions & Information" footer panel.
  2. Export Data: Download your daily tracking data as a CSV, PDF, or send it to a printer.
  3. Data History:
    • Preserve Current Month: Opens a dialog to choose how to save the data.
      • Temporary Snapshot: Takes a snapshot of the current month's performance and saves it to history. Your current data remains unchanged.
      • Finalize and Reset Month: Takes a final snapshot, saves it to history, and then clears the `Holidays`, `Total Target` for all KPIs, and all daily entry data to prepare for the next tracking month. This is a destructive action.
    • View History: Opens a window to view all your saved monthly snapshots.
  4. Administration & Security:
    • Set Password: Enter a password in the input box and click this button to protect your tracker. When you set a password, the Tracker will give you a one-time recovery code. Save it in a safe place!
    • Lock / Unlock: Click Lock to make all inputs read-only, except for today's data entry. Click Unlock (and enter your password if set) to make everything editable again.
    • Reset All: This will permanently delete all data and settings. It will ask for your password if one is set.
    • Forgot Password?: If you forget your password, click this. You will be prompted to enter the recovery code you saved.
      • Last Resort Recovery: If you have also lost your recovery code, this process offers a final option. The Tracker will ask you to provide the exact Start Date and End Date of the current tracking period. Save these dates in a safe place! If the dates you provide match what is saved in the tracker, your password will be removed.
  5. About this Tool:
    • View User Manual: Opens this guide.

Remember: All configuration changes, data entry, and even the collapsed state of cards are saved automatically as you work!
