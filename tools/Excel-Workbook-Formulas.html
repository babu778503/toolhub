✅ Core Workbook / Reference Functions

ISREF

Purpose:Checks if a value is a valid cell reference.
Return value:TRUE or FALSE
Syntax:ISREF(value)
Examples:=ISREF(A1) // TRUE
=ISREF("Text") // FALSE

HYPERLINK

Purpose:Creates a shortcut or jump that opens a document stored on a network server, an intranet, or the internet.
Return value:Clickable link text
Syntax:HYPERLINK(link_location, [friendly_name])
Examples:=HYPERLINK("https://toolshub365.com", "Tools Hub")

✅ Information Functions

ISBLANK

Purpose:Checks whether a reference is to an empty cell.
Return value:TRUE or FALSE
Syntax:ISBLANK(value)
Examples:=ISBLANK(A1)

ISNUMBER

Purpose:Checks whether a value is a number.
Return value:TRUE or FALSE
Syntax:ISNUMBER(value)
Examples:=ISNUMBER(123) // TRUE

ISTEXT

Purpose:Checks whether a value is text.
Return value:TRUE or FALSE
Syntax:ISTEXT(value)
Examples:=ISTEXT("Hello") // TRUE

ISLOGICAL

Purpose:Checks whether a value is a logical value (TRUE or FALSE).
Return value:TRUE or FALSE
Syntax:ISLOGICAL(value)
Examples:=ISLOGICAL(TRUE) // TRUE

ISERROR

Purpose:Checks whether a value is any error.
Return value:TRUE or FALSE
Syntax:ISERROR(value)
Examples:=ISERROR(A1/0) // TRUE

ISNA

Purpose:Checks whether a value is specifically the #N/A error.
Return value:TRUE or FALSE
Syntax:ISNA(value)
Examples:=ISNA(VLOOKUP(...))

ISERR

Purpose:Checks whether a value is any error EXCEPT #N/A.
Return value:TRUE or FALSE
Syntax:ISERR(value)
Examples:=ISERR(A1)

ISFORMULA

Purpose:Checks whether a reference is to a cell containing a formula.
Return value:TRUE or FALSE
Syntax:ISFORMULA(reference)
Examples:=ISFORMULA(B2)

ISNONTEXT

Purpose:Checks whether a value is not text (blank cells are not text).
Return value:TRUE or FALSE
Syntax:ISNONTEXT(value)
Examples:=ISNONTEXT(123) // TRUE

N

Purpose:Returns a value converted to a number. Text becomes 0, TRUE becomes 1.
Return value:Numeric value
Syntax:N(value)
Examples:=N(TRUE) // 1
=N("Text") // 0

TYPE

Purpose:Returns an integer representing the data type of a value (1=number, 2=text, 4=logical, 16=error, 64=array).
Return value:Integer
Syntax:TYPE(value)
Examples:=TYPE("Hello") // 2
=TYPE(100) // 1

✅ Cell & Reference Functions

CELL

Purpose:Returns information about the formatting, location, or contents of a cell.
Return value:Text or Number (based on info type)
Syntax:CELL(info_type, [reference])
Examples:=CELL("address", A1) // "$A$1"
=CELL("type", B2)

INFO

Purpose:Returns information about the current operating environment.
Return value:Text environment info
Syntax:INFO(type_text)
Examples:=INFO("system") // "pcdos" or "mac"
=INFO("osversion")

FORMULATEXT

Purpose:Returns the formula of a referenced cell as a text string.
Return value:Text string
Syntax:FORMULATEXT(reference)
Examples:=FORMULATEXT(A1) // "=SUM(B1:B10)"

ADDRESS

Purpose:Creates a cell reference as text, given specified row and column numbers.
Return value:Text (Cell Reference)
Syntax:ADDRESS(row_num, column_num, [abs_num])
Examples:=ADDRESS(1, 1) // "$A$1"
=ADDRESS(1, 1, 4) // "A1"

INDIRECT

Purpose:Returns the valid cell reference specified by a text string.
Return value:Cell contents/Reference
Syntax:INDIRECT(ref_text, [a1])
Examples:=INDIRECT("A" & 5) // Value in A5
=INDIRECT("Sheet2!B1")

OFFSET

Purpose:Returns a reference to a range that is a specified number of rows/columns from a cell.
Return value:Cell Reference / Range
Syntax:OFFSET(reference, rows, cols, [height], [width])
Examples:=OFFSET(A1, 1, 1) // Returns B2

✅ Row & Column Functions

ROW

Purpose:Returns the row number of a given cell reference.
Return value:Integer
Syntax:ROW([reference])
Examples:=ROW(C5) // 5
=ROW() // Current row

ROWS

Purpose:Returns the total number of rows in a specified reference or array.
Return value:Integer
Syntax:ROWS(array)
Examples:=ROWS(A1:B10) // 10

COLUMN

Purpose:Returns the column number of a given cell reference.
Return value:Integer
Syntax:COLUMN([reference])
Examples:=COLUMN(C5) // 3
=COLUMN() // Current col

COLUMNS

Purpose:Returns the total number of columns in a specified reference or array.
Return value:Integer
Syntax:COLUMNS(array)
Examples:=COLUMNS(A1:C10) // 3

✅ Lookup & Reference (Workbook Level)

INDEX

Purpose:Returns a value or reference of the cell at the intersection of a row and column in a given range.
Return value:Value / Reference
Syntax:INDEX(array, row_num, [column_num])
Examples:=INDEX(A1:C10, 2, 3)

MATCH

Purpose:Searches for a specified item in a range of cells and returns its relative position.
Return value:Integer (Position)
Syntax:MATCH(lookup_value, lookup_array, [match_type])
Examples:=MATCH("Apple", A1:A10, 0)

XMATCH

Purpose:Searches for a specified item and returns its relative position. Defaults to exact match.
Return value:Integer (Position)
Syntax:XMATCH(lookup_value, lookup_array, [match_mode])
Examples:=XMATCH("Apple", A1:A10)

XLOOKUP

Purpose:Modern and flexible lookup function to search a range and return the corresponding item.
Return value:Value / Array
Syntax:XLOOKUP(lookup_value, lookup_array, return_array)
Examples:=XLOOKUP(D2, A1:A10, B1:B10)

✅ Dynamic Array Reference (Excel 365)

FILTER

Purpose:Filters a range of data based on criteria you define, returning an array.
Return value:Spilled Array
Syntax:FILTER(array, include, [if_empty])
Examples:=FILTER(A2:B10, B2:B10>50, "None")

SORT

Purpose:Sorts the contents of a range or array automatically.
Return value:Spilled Array
Syntax:SORT(array, [sort_index], [sort_order])
Examples:=SORT(A2:B10, 2, -1) // Descending by col 2

SORTBY

Purpose:Sorts a range or array based on the values in an external corresponding array.
Return value:Spilled Array
Syntax:SORTBY(array, by_array1, [sort_order1])
Examples:=SORTBY(A2:A10, B2:B10, 1)

UNIQUE

Purpose:Returns a list of unique values from a range or array.
Return value:Spilled Array
Syntax:UNIQUE(array, [by_col], [exactly_once])
Examples:=UNIQUE(A2:A20)

✅ Logical & Error Handling

IF

Purpose:Performs a logical test and returns one value for a TRUE result, and another for FALSE.
Return value:Defined Value
Syntax:IF(logical_test, value_if_true, [value_if_false])
Examples:=IF(A1>10, "Yes", "No")

IFERROR

Purpose:Catches errors and returns a specified clean value instead of an error code.
Return value:Value or Error Replacement
Syntax:IFERROR(value, value_if_error)
Examples:=IFERROR(A1/B1, 0)

IFNA

Purpose:Specifically handles #N/A errors, useful when VLOOKUP/XLOOKUP finds no match.
Return value:Value or #N/A Replacement
Syntax:IFNA(value, value_if_na)
Examples:=IFNA(VLOOKUP(D2,A:B,2,0), "Not Found")

✅ Workbook Linking & Navigation

HYPERLINK (Internal Links)

Purpose:Creates a clickable button/text to navigate to another sheet or cell in the same workbook.
Return value:Clickable Link
Syntax:HYPERLINK("#"&"SheetName!Cell", [name])
Examples:=HYPERLINK("#Sheet2!A1", "Go to Sheet2")

INDIRECT (External Workbook Reference)

Purpose:Dynamically references ranges in other open workbooks using text strings.
Return value:Cell Value
Syntax:INDIRECT("'["&book&"]"&sheet&"'!"&cell)
Examples:=INDIRECT("'[Data.xlsx]Q1'!A1")

CELL ("filename")

Purpose:Returns the full path, workbook name, and sheet name of the specified reference.
Return value:File Path String
Syntax:CELL("filename", [reference])
Examples:=CELL("filename", A1)

INFO ("directory")

Purpose:Returns the path of the current directory/folder context.
Return value:Path String
Syntax:INFO("directory")
Examples:=INFO("directory")

✅ Advanced Reference Techniques

CHOOSE

Purpose:Returns a value from a list using a given index number.
Return value:Selected Value / Reference
Syntax:CHOOSE(index_num, value1, [value2], ...)
Examples:=CHOOSE(2, "Red", "Blue", "Green") // "Blue"

SWITCH

Purpose:Evaluates an expression against a list of values and returns the first matching result.
Return value:Matched Value
Syntax:SWITCH(expression, val1, res1, [default])
Examples:=SWITCH(A1, 1, "Mon", 2, "Tue", "Unknown")

LET

Purpose:Assigns names to calculation results. Speeds up formulas by calculating intermediate steps once.
Return value:Calculated Result
Syntax:LET(name1, name_val1, calculation)
Examples:=LET(x, SUM(A1:A10), IF(x>100, x, 0))

LAMBDA

Purpose:Creates custom, reusable functions without VBA.
Return value:Function Result
Syntax:LAMBDA([parameter1, ...], calculation)
Examples:=LAMBDA(x, y, x*y)(5, 2) // Returns 10

🔥 Bonus Workbook Techniques

Dynamic Named Range (OFFSET)

Purpose:Creates a range that grows automatically as new data is added.
Return value:Dynamic Array Range
Syntax:OFFSET(start, 0, 0, COUNTA(col), 1)
Examples:=OFFSET(Sheet1!$A$1, 0, 0, COUNTA(Sheet1!$A:$A), 1)

Sheet Reference (INDIRECT)

Purpose:Uses a cell's text value to refer to a specific worksheet dynamically.
Return value:Cell Value from Sheet
Syntax:INDIRECT("'" & sheet_cell & "'!A1")
Examples:=INDIRECT("'" & B1 & "'!C5")

External Workbook Link (INDIRECT)

Purpose:Builds an external file reference via formula concatenation (requires file to be open).
Return value:Data Value
Syntax:INDIRECT("'["&book_val&"]"&sheet_val&"'!A1")
Examples:=INDIRECT("'["&A1&".xlsx]Summary'!B2")

Get File Path (CELL)

Purpose:Extracts the exact folder path of the active workbook using text manipulation.
Return value:Folder Path String
Syntax:LEFT(CELL, FIND("[", CELL)-1)
Examples:=LEFT(CELL("filename", A1), FIND("[", CELL("filename", A1))-1)

Clickable Navigation (HYPERLINK)

Purpose:Builds dynamic index or Table of Contents buttons.
Return value:Active URL/Jump
Syntax:HYPERLINK("#'" & cell & "'!A1", "Go")
Examples:=HYPERLINK("#'" & A2 & "'!A1", "Go to " & A2)
