Dishwasher Tips

  • Scrape off excess food but don't pre-rinse dishes
  • Load dishes with dirty sides facing the center
  • Use the appropriate detergent for your water hardness
  • Run full loads for maximum efficiency
  • Clean the filter regularly for optimal performance
  • Use the heated dry option only when necessary
  • Check for recalls and update firmware if available
